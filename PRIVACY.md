# Privacy Policy - Intelligent Tab & Session Manager

**Last Updated:** January 2025

## 1. Information We Collect

Intelligent Tab & Session Manager collects the following information to provide its services:

### 1.1 Session Data
- Session names and tags
- Tab URLs and titles
- Tab group configurations
- Session creation and modification timestamps

### 1.2 User Information
- User email address (for Drive access)
- User preferences and settings

### 1.3 Configuration Data
- Auto-save settings and intervals
- Template definitions
- Workspace layouts

### 1.4 Activity Logs
- Timestamps of session saves and restores
- Template applications
- Auto-save operations

## 2. How We Use Your Information

We use your information to:
- Save and restore browser session states
- Organize tabs into groups
- Apply workspace templates
- Track usage patterns
- Provide session backups

## 3. Data Storage and Retention

### 3.1 Storage Location
- All configuration data is stored in Google Apps Script PropertiesService
- Session data is stored in user properties
- Session backups are stored in Google Drive

### 3.2 Data Retention
- Configuration data is retained until you delete it
- Activity logs are limited to the most recent 100 entries
- Session data remains until you delete it

## 4. Data Sharing

We do not sell, rent, or share your data with third parties for marketing purposes.

### 4.1 Limited Sharing
- No data sharing occurs
- All data remains within your Google account

## 5. Data Security

We implement appropriate security measures:
- All data is stored within Google's secure infrastructure
- OAuth 2.0 authentication for Google API access
- Drive access is limited to files you create
- No external servers or databases

## 6. Your Rights

You have the right to:
- Access your session data and settings
- Delete sessions and configuration data
- Clear activity logs
- Uninstall the add-on at any time

## 7. Children's Privacy

Intelligent Tab & Session Manager is not directed to children under 13. We do not knowingly collect personal information from children under 13.

## 8. Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new policy on this page.
